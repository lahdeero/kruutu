# kruutu


## Introduction

Switch between display setups fast.

Screens: DP-1, DVI-D-1, HDMI-A-1

1. DVI-D-1, DP-1 (extend). DP-1 Is main display on right hand side
2. DP-1, HDMI-A-1 (mirror)
3. DP-1
4. HDMI-A-1, DVI-D-1, DP-1. HDMI-A-1 left side, DVI-D-1 middle and HDMI-A-1 right

## Requirements

kscreen-doctor

## Installation

```bash
git clone https://github.com/lahdeero/kruutu.git
cd kruutu
cp kruutu /usr/local/bin
```

## Usage

```bash
kruutu <number>
```

```bash
kruutu 2
```
